# Hermes 智能体

由 [Nous Research](https://nousresearch.com) 开发的自我改进 AI 智能体。这是唯一一个内置学习循环的智能体——它能从经验中创建技能，在使用过程中改进技能，主动推动自己持久化知识，并能在会话间不断加深对您的理解模型。

## 安装

**Linux / macOS / WSL2**

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

**Windows（原生，PowerShell）** — *早期测试版，[详情 →](/user-guide/windows-native)*

```powershell
iex (irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1)
```

**Android (Termux)** — 与 Linux 使用相同的 curl 单行命令；安装程序会自动检测 Termux。

请参阅完整的 **[安装指南](/getting-started/installation)**，了解安装程序的具体操作、每用户与 root 的布局，以及 Windows 相关说明。

## 什么是 Hermes 智能体？

它不是绑定在 IDE 上的编码助手，也不是围绕单个 API 的聊天机器人封装。它是一个**自主智能体**，运行时间越长，能力就越强。它可以驻留在您放置它的任何地方——一个 5 美元的 VPS、一个 GPU 集群，或者一个闲置时几乎不花费成本的无服务器基础设施（Daytona、Modal）。您可以从 Telegram 与它对话，而它正在一个您从未 SSH 登录过的云虚拟机上工作。它不受限于您的笔记本电脑。

## 核心特性

- **闭环学习循环** — 智能体策划的记忆与定期提示、自主创建技能、使用过程中的技能自我改进、带有 LLM 总结的 FTS5 跨会话回忆，以及 [Honcho](https://github.com/plastic-labs/honcho) 辩证用户建模
- **随处运行，不限于笔记本电脑** — 6 种终端后端：本地、Docker、SSH、Daytona、Singularity、Modal。Daytona 和 Modal 提供无服务器持久化——您的环境在闲置时休眠，成本几乎为零
- **与您同在** — CLI、Telegram、Discord、Slack、WhatsApp、Signal、Matrix、Mattermost、Email、SMS、钉钉、飞书、企业微信、微信、QQ 机器人、元宝、BlueBubbles、Home Assistant、Microsoft Teams、Google Chat 等——一个网关支持 20 多个平台
- **由模型训练者打造** — 由 [Nous Research](https://nousresearch.com) 创建，Hermes、Nomos 和 Psyche 模型背后的实验室。支持 [Nous Portal](https://portal.nousresearch.com)、[OpenRouter](https://openrouter.ai)、OpenAI 或任何端点
- **定时自动化** — 内置 cron 任务，可向任何平台推送
- **委派与并行** — 生成隔离的子智能体以处理并行工作流。通过 `execute_code` 进行编程式工具调用，可将多步骤管道压缩为单次推理调用
- **开放标准技能** — 兼容 [agentskills.io](https://agentskills.io)。技能可移植、可共享，并通过技能中心由社区贡献
- **全面的网络控制** — 搜索、提取、浏览、视觉、图像生成、文本转语音
- **MCP 支持** — 连接到任何 MCP 服务器以扩展工具能力
- **研究就绪** — 批量处理、轨迹导出、使用 Atropos 进行 RL 训练。由 [Nous Research](https://nousresearch.com) 开发——Hermes、Nomos 和 Psyche 模型背后的实验室
