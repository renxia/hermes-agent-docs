## 概述
本技能教导智能体构建机构级的可比公司分析，该分析结合了经营指标、估值倍数和统计基准测试。输出是一个结构化的 Excel/电子表格，可通过同行比较支持明智的投资决策。

**参考资料与情境化：**

在 `examples/comps_example.xlsx` 中提供了一个可比公司分析示例。在使用此技能目录中的此示例文件或其他示例文件时，请智能地使用它们：

**不应使用示例来：**

**始终首先问自己：**

2.  **“受众是谁？”** （投资委员会、董事会演示、快速参考、详细备忘录）

**根据具体情况调整：**

- **公司熟悉度**：知名公司可能需要较少背景介绍，更侧重于变化分析
- **决策类型**：并购与持续的组合监控需要不同的侧重点

**核心原则：** 使用模板原则（清晰的结构、统计严谨性、透明的公式），但根据上下文改变执行方式。目标是机构级质量的分析，而非机构外观的模板。

用户提供的示例和明确偏好始终优先于默认设置。

## 核心理念
**“先建立正确的结构，然后让数据自己讲述故事。”**

从能促使对重要内容进行战略思考的标题开始，输入干净的数据，构建透明的公式，让统计数据自动生成。一个好的可比分析应能让未参与构建的人立即读懂。

---

## ⚠️ 关键：公式优先于硬编码 + 分步验证

**公式，而非硬编码：**
- 每个衍生值（利润率、倍数、统计量）必须是引用输入单元格的 Excel 公式 —— 绝不能是粘贴进来的预计算数字
- 使用 Python/openpyxl 构建工作表时：写 `cell.value = "=E7/C7"` (公式字符串)，而非 `cell.value = 0.687` (计算结果)
- 唯一可以硬编码的值应是原始输入数据（收入、EBITDA、股价等）—— 且每个都应在单元格批注中注明来源
- 原因：当输入数据更改时，模型必须能自动更新。硬编码的利润率是一个等待发生的沉默错误。

**与用户分步验证：**
- 设置好结构后 → 向用户展示标题布局，然后再填充数据
- 输入原始数据后 → 向用户展示输入区块，并在构建公式前确认来源/期间
- 构建经营指标公式后 → 向用户展示计算出的利润率，并在转向估值前进行合理性检查
- 构建估值倍数后 → 向用户展示倍数，并在添加统计数据前确认它们看起来合理
- **不要**从头到尾构建整个工作表然后才呈现 —— 通过确认每个部分来尽早发现错误

## 第 1 部分：文档结构与设置

### 标题区块（第 1-3 行）
```
第 1 行：[分析标题] - 可比公司分析
第 2 行：[包含股票代码的公司列表] • [公司 1 (TICK1)] • [公司 2 (TICK2)] • [公司 3 (TICK3)]
第 3 行：截至 [期间] | 除每股金额和比率外，所有数字单位为 [百万/十亿美元 (USD)]
```

**原因：** 立即建立上下文。任何打开此文件的人都知道他们正在查看什么、创建时间以及如何解读这些数字。

### 视觉惯例标准（可选 - 用户偏好和上传的模板始终优先）

**重要提示：这些仅为建议默认值。始终优先考虑：**
1. 用户明确的格式偏好
2. 任何上传模板文件的格式
3. 公司/团队的风格指南
4. 这些默认值（仅在没有其他指导时使用）

**建议字体与排版：**
- **字体系列**：Times New Roman（专业、易读、行业标准）
- **字号**：数据单元格 11pt，标题 12pt
- **粗体文本**：节标题、公司名称、统计标签

**默认颜色与底纹 - 专业蓝/灰色调（越简洁越好）：**
- **保持克制** —— 仅使用蓝色和灰色。**不要**引入绿色、橙色、红色或多种强调色。一个简洁的可比分析表总共只使用 3-4 种颜色。
- **节标题**（例如，“经营统计与财务指标”）：
    - 深蓝色背景（`#1F4E79` 或 `#17365D` 藏青色）
    - 白色粗体文字
    - 跨所有列的整行底纹
- **列标题**（例如，“公司”、“收入”、“利润率”）：
    - 浅蓝色背景（`#D9E1F2` 或类似浅蓝）
    - 黑色粗体文字
    - 居中对齐
- **数据行**：
    - 公司数据使用白色背景
    - 公式的文字使用黑色；硬编码输入的文字使用蓝色
- **统计行**（最大值、第 75 百分位数等）：
    - 浅灰色背景（`#F2F2F2`）
    - 黑色文字，标签左对齐
- **整个调色板就是这些**：深蓝 + 浅蓝 + 浅灰 + 白色。除非用户的模板另有说明，否则不使用其他颜色。

**建议格式惯例：**
- **小数精度**：
    - 百分比：1 位小数（12.3%）
    - 倍数：1 位小数（13.5x）
    - 美元金额：无小数，使用千位分隔符（69,632）
    - 显示为百分比的利润率：1 位小数（68.7%）
- **边框**：无边框（简洁、极简外观）
- **对齐**：所有指标居中对齐，以实现整洁统一的外观
- **单元格尺寸**：所有列宽应均匀/一致，所有行高应一致（创建整洁、专业的网格）

**注意：** 如果用户提供模板文件或指定不同的格式，请使用该格式。

## 第 2 部分：经营统计与财务指标

### 核心列（从这些开始）
1.  **公司** - 名称保持格式一致
2.  **收入** - 规模指标（根据上下文可以是 LTM、季度或年度）
3.  **收入增长率** - 同比百分比变化
4.  **毛利润** - 收入减去销货成本
5.  **毛利率** - 毛利润/收入（基本盈利能力）
6.  **EBITDA** - 息税折旧摊销前利润
7.  **EBITDA 利润率** - EBITDA/收入（经营效率）

### 可选项（根据行业/目的选择）
- **季度 vs LTM** - 如果季节性很重要，两者都包含
- **自由现金流** - 适用于资本密集型或 SaaS 企业
- **自由现金流利润率** - 自由现金流/收入（现金生成效率）
- **净利润** - 适用于成熟、盈利的公司
- **营业利润** - 适用于折旧摊销变化较大的企业
- **资本支出指标** - 适用于资产密集型行业
- **Rule of 40** - 专门适用于 SaaS（增长率 % + 利润率 %）
- **自由现金流转化率** - 用于盈余质量分析（高级）

### 公式示例（以第 7 行为例）
```excel
// 核心比率 - 这些总是计算的
毛利率 (F7): =E7/C7
EBITDA 利润率 (H7): =G7/C7

// 可选比率 - 如果相关则包含
自由现金流利润率: =[自由现金流]/[收入]
净利率: =[净利润]/[收入]
Rule of 40: =[增长率 %]+[自由现金流利润率 %]
```

**黄金法则：** 每个比率都应是 [某项] / [收入] 或 [本表中的某项] / [某项]。保持简单。

### 统计区块（在公司数据之后）

**关键：为所有可比指标（比率、利润率、增长率、倍数）添加统计公式。**

```
[留一个空行用于视觉分隔]
- 最大值: =MAX(B7:B9)
- 第 75 百分位数: =QUARTILE(B7:B9,3)
- 中位数: =MEDIAN(B7:B9)
- 第 25 百分位数: =QUARTILE(B7:B9,1)
- 最小值: =MIN(B7:B9)
```

**需要统计的列（可比指标）：**
- 收入增长率 %、毛利率 %、EBITDA 利润率 %、每股收益
- EV/收入、EV/EBITDA、市盈率、股息收益率 %、贝塔系数

**不需要统计的列（规模指标）：**
- 收入、EBITDA、净利润（绝对规模因公司而异）
- 市值、企业价值（在不同规模的公司间不可比）

**注意：** 在公司数据和统计行之间添加一个空行用于视觉分隔。**不要**添加“行业统计”或“估值统计”标题行。

**四分位数为何重要：** 它们展示分布情况，而不仅仅是平均值。第 75 百分位数的倍数告诉你“溢价”公司的交易水平。

## 第三部分：估值倍数与投资指标

### 核心估值列（从此开始）
1. **公司** - 顺序与运营部分相同
2. **市值** - 当前市场估值
3. **企业价值** - 市值 ± 净债务/现金
4. **EV/收入** - 市场为每1美元销售额支付的价格
5. **EV/EBITDA** - 市场为每1美元盈利支付的价格
6. **市盈率** - 股价相对于净收益

### 可选估值指标（根据上下文选择）
- **自由现金流收益率** - 自由现金流/市值（用于以现金为核心的分析）
- **PEG比率** - 市盈率/增长率（用于增长型公司）
- **市净率** - 市值 vs. 账面价值（用于资产密集型企业）
- **ROE/ROA** - 回报率指标（用于盈利能力比较）
- **收入/EBITDA复合年增长率** - 历史增长率（用于趋势分析）
- **资产周转率** - 收入/资产（用于运营效率分析）
- **债务权益比** - 杠杆率（用于资本结构分析）

**关键原则：** 包含对您所在行业重要的3-5个核心倍数。不要仅仅因为可以就包含所有可能的指标。

### 公式示例
```excel
// 核心倍数 - 始终包含这些
EV/收入： =[企业价值]/[最近12个月收入]
EV/EBITDA： =[企业价值]/[最近12个月EBITDA]
市盈率： =[市值]/[净收入]

// 可选倍数 - 如果数据可用则包含
自由现金流收益率： =[最近12个月自由现金流]/[市值]
PEG比率： =[市盈率]/[增长率 %]
```

### 交叉引用规则
**关键：** 估值倍数必须引用运营指标部分。永远不要输入两次相同的原始数据。如果收入在C7单元格，那么EV/收入的公式应该引用C7。

### 统计数据块
结构与运营部分相同：每个指标都需列出最大值、第75百分位数、中位数、第25百分位数、最小值。在公司数据和统计数据之间添加一个空行以进行视觉分隔。**不要**添加“估值统计”标题行。

## 第四部分：注释与方法论说明

### 必需组成部分

**数据来源与质量：**
- 数据来自哪里？（标普 Kensho MCP、FactSet MCP、Daloopa MCP、彭博、美国证交会文件）
- 它涵盖什么时间段？（2024年第四季度，经审计数据）
- 如何进行验证？（与10-K/10-Q报告交叉核对）
- 注意：如果可用，请优先使用MCP数据来源（标普 Kensho、FactSet、Daloopa），以获得更好的准确性和可追溯性

**关键定义：**
- EBITDA计算方法（毛利润 + 折旧摊销，或营业利润 + 折旧摊销）
- 自由现金流公式（经营现金流 - 资本支出）
- 特殊指标说明（40规则、自由现金流转化率）
- 时间段定义（最近12个月、复合年增长率计算期）

**估值方法论：**
- 企业价值是如何计算的？（市值 + 净债务）
- 使用了哪些增长率？（历史复合年增长率、远期估计值）
- 是否进行了任何调整？（排除一次性项目、正常化利润率）

**分析框架：**
- 投资主题是什么？（云/SaaS效率）
- 哪些指标最重要？（现金产生能力、资本效率）
- 读者应如何解读统计数据？（四分位数提供上下文）

## 第五部分：选择正确的指标（决策框架）

### 从 "我要回答什么问题？" 开始

**"哪家公司价值被低估？"**
→ 关注：企业价值/收入、企业价值/息税折旧摊销前利润、市盈率、市值
→ 忽略：运营细节、增长指标

**"哪家公司效率最高？"**
→ 关注：毛利率、息税折旧摊销前利润率、自由现金流利润率、资产周转率
→ 忽略：规模指标、绝对美元金额

**"哪家公司增长最快？"**
→ 关注：收入增长率、息税折旧摊销前利润复合年增长率、用户/客户增长
→ 忽略：利润率指标、杠杆率

**"哪家是最好的现金生成器？"**
→ 关注：自由现金流、自由现金流利润率、自由现金流转化率、资本支出强度
→ 忽略：息税折旧摊销前利润、市盈率

### 行业特定指标选择

**软件/SaaS：**
必须拥有：收入增长、毛利率、40法则
可选：年度经常性收入、净美元留存率、获客成本回收期
忽略：资产周转率、库存指标

**制造业/工业：**
必须拥有：息税折旧摊销前利润率、资产周转率、资本支出/收入
可选：资产回报率、存货周转率、订单积压
忽略：40法则、SaaS指标

**金融服务：**
必须拥有：净资产收益率、资产回报率、效率比率、市盈率
可选：净息差、贷款损失准备金
忽略：毛利率、息税折旧摊销前利润（对银行意义不大）

**零售/电子商务：**
必须拥有：收入增长、毛利率、存货周转率
可选：同店销售额、客户获取成本
忽略：重度研发或资本支出指标

### "5-10法则"

**5个运营指标** - 收入、增长、2-3个利润率/效率指标
**5个估值指标** - 市值、企业价值、3个倍数
**= 总共10列** - 足以说明情况，又不会多到让人迷失重点

如果你的指标超过15个，很可能包含了噪音。要果断删减。

## 第六部分：最佳实践与质量检查

### 开始前准备
1. **定义可比公司组** - 公司必须真正可比（相似的业务模式、规模、地域）
2. **选择正确的时期** - 最近十二个月数据可平滑季节性；季度数据能显示趋势
3. **提前统一单位** - 百万与十亿的决定影响一切
4. **映射数据来源** - 了解每个数字的来源

### 构建过程中
1. **先输入所有原始数据** - 在编写公式前完成蓝色文本部分
2. **为所有硬编码输入添加单元格批注** - 右键单击单元格 → 插入批注 → 记录来源或假设

   **对于有来源的数据，精确注明出处：**
   - 示例："彭博终端 - MSFT股票DES，访问于2024-10-02"
   - 示例："2024年第四季度10-K文件，第42页，行项目'总收入'"
   - 示例："截至2024-10-02的FactSet共识预期"
   - **尽可能包含超链接**：右键单击单元格 → 链接 → 粘贴SEC文件、数据来源或报告的URL

   **对于假设，解释推理过程：**
   - 示例："假设息税折旧摊销前利润率为15%，基于同行中位数，公司未披露"
   - 示例："估计企业价值为市值+5000万美元净债务（来自第三季度资产负债表，第四季度数据尚未公布）"
   - 示例："远期市盈率基于华尔街共识每股收益3.45美元（12位分析师估计的平均值）"

   **为何重要**：实现审计追踪、数据验证、假设透明化以及未来更新
3. **逐行构建公式** - 移动到下一项前测试每个计算
4. **为标题使用绝对引用** - $C$6 锁定标题行
5. **保持格式一致** - 百分数按百分数格式，而非小数
6. **添加条件格式** - 自动高亮异常值

### 合理性检查
- **利润率测试**：毛利率 > 息税折旧摊销前利润率 > 净利润率（按定义始终成立）
- **倍数合理性**：
  - 企业价值/收入：通常0.5-20倍（行业差异大）
  - 企业价值/息税折旧摊销前利润：通常8-25倍（行业间相对一致）
  - 市盈率：通常10-50倍（取决于增长率）
- **增长-倍数相关性**：增长越高通常意味着倍数越高
- **规模-效率权衡**：大公司通常利润率更高（规模效益）

### 应避免的常见错误
❌ 在公式中混用市值和企业价值
❌ 分子和分母使用不同时间段（最近十二个月 vs 季度）
❌ 将数字硬编码到公式中，而非使用单元格引用
❌ **硬编码输入没有单元格批注注明来源或解释假设**
❌ 有可用SEC文件或数据来源时缺少超链接
❌ 包含太多指标而没有明确目的
❌ 包含不可比公司（业务模式不同）
❌ 使用过时数据而未披露
❌ 错误计算百分比平均值（应为中位数）

## 第八部分：工作流程与实用技巧

### 分步流程
1. **建立结构**（30分钟）
   - 创建所有标题
   - 格式化单元格（蓝色用于输入，黑色用于公式）
   - 确定单位和日期引用

2. **收集数据**（60-90分钟）
   - 从主要来源获取（S&P Kensho MCP、FactSet MCP、如可用Daloopa MCP；否则彭博、SEC）
   - 在蓝色单元格中输入所有原始数字
   - 在备注区记录来源

3. **构建公式**（30分钟）
   - 从简单比率开始（利润率）
   - 进展到倍数（企业价值/收入）
   - 添加交叉检查（利润率合理吗？）

4. **添加统计**（15分钟）
   - 为所有列复制公式结构
   - 验证范围正确（B7:B9，而非B7:B10）
   - 检查四分位数逻辑

5. **质量控制**（30分钟）
   - 运行合理性检查
   - 验证公式引用
   - 检查#DIV/0!或#REF!错误
   - 与已知基准对比

6. **文档化**（15分钟）
   - 完成备注区
   - 添加数据来源
   - 定义方法论
   - 为分析加盖日期戳

## 第10节：危险信号与警告标志

### 数据质量问题
🚩 时间段不一致（混合使用季度和年度数据）  
🚩 缺失数据且无解释  
🚩 数据来源之间差异显著（差异>10%）

### 估值危险信号
🚩 对负EBITDA公司使用EBITDA倍数估值（应使用营收倍数）  
🚩 市盈率>100x且无超高增长故事支撑  
🚩 毛利率对该行业而言不合逻辑

### 可比性问题
🚩 财年结束日不同（导致时间错配）  
🚩 混合纯业务公司和综合性集团  
🚩 将商业模式实质性不同的公司标记为"可比公司"

**当有疑问时，排除该公司。** 拥有3家完美的可比公司好过6家有问题的公司。

## 第11节：公式参考指南

### 基本Excel公式
```excel
// 统计函数
=AVERAGE(range)          // 简单平均值
=MEDIAN(range)           // 中位数
=QUARTILE(range, 1)      // 第25百分位数
=QUARTILE(range, 3)      // 第75百分位数
=MAX(range)              // 最大值
=MIN(range)              // 最小值
=STDEV.P(range)          // 标准差

// 财务计算
=B7/C7                   // 简单比率（利润率）
=SUM(B7:B9)/3            // 多家公司的平均值
=IF(B7>0, C7/B7, "N/A")  // 条件计算
=IFERROR(C7/D7, 0)       // 处理除零错误

// 跨表引用
='Sheet1'!B7             // 引用另一个工作表
=VLOOKUP(A7, Table1, 2)  // 从数据表中查找
=INDEX(MATCH())          // 高级查找

// 格式化
=TEXT(B7, "0.0%")        // 格式化为百分比
=TEXT(C7, "#,##0")       // 千位分隔符
```

### 常用比率公式
```excel
毛利率 = 毛利润 / 营收
EBITDA利润率 = EBITDA / 营收
自由现金流利润率 = 自由现金流 / 营收
自由现金流转化率 = 自由现金流 / 经营现金流
股本回报率 = 净收入 / 股东权益
资产回报率 = 净收入 / 总资产
资产周转率 = 营收 / 总资产
债务权益比 = 总债务 / 股东权益
```

## 关键原则总结

1.  **结构驱动洞察** – 正确的标题引导正确的思考
2.  **少即是多** – 5-10个关键指标胜过20个不相关的指标
3.  **为你的问题选择指标** – 估值分析 ≠ 效率分析
4.  **统计揭示模式** – 中位数/四分位数比平均数更能说明问题
5.  **透明胜于复杂** – 简单的公式大家都能理解
6.  **可比性是王道** – 排除劣质可比公司好过强行凑数
7.  **记录你的选择** – 在注释部分解释选择哪些指标及原因
