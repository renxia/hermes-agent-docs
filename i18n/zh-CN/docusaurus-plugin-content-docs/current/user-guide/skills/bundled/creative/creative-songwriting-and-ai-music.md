# 歌曲创作与 AI 音乐生成

此处所有内容均为**指导原则**，而非规则。艺术本就是为了打破规则而生。  
使用对歌曲有用的部分，忽略无用的部分。

---

## 1. 歌曲结构（任选一种或自创）

常见结构骨架 —— 可混合、修改或完全舍弃：

```
ABABCB  主歌/副歌/主歌/副歌/桥段/副歌    （流行/摇滚最常见）
AABA    主歌/主歌/桥段/主歌（基于叠句）    （爵士标准曲、民谣 ballad）
ABAB    主歌/副歌交替                    （简洁直接）
AAA     主歌/主歌/主歌（分节歌，无副歌）     （民谣、叙事歌曲）
```

六大组成部分：
- 前奏（Intro）      —— 奠定情绪，吸引听众
- 主歌（Verse）      —— 讲述故事、细节、构建世界
- 预副歌（Pre-Chorus）—— 可选，在副歌前制造张力
- 副歌（Chorus）     —— 情感核心，令人记忆深刻的部分
- 桥段（Bridge）     —— 转折，视角或调性的变化
- 尾奏（Outro）      —— 告别，可呼应或颠覆前文

你不必全部使用。有些伟大的歌曲只有一个段落，但不断演变。  
结构服务于情感，而非反之。

## 2. 押韵、格律与声音

**押韵类型**（从紧密到松散）：
- 完全押韵：lean/mean
- 家族押韵：crate/braid
- 元音押韵（Assonance）：had/glass（元音相同，结尾不同）
- 辅音押韵（Consonance）：scene/when（元音不同，结尾相似）
- 近似/斜押韵：足以暗示关联，但不过度束缚

混合使用。全部使用完全押韵会像童谣，  
全部使用斜押韵则显得懒惰。关键在于平衡。

**句内押韵**：在一行内押韵，而非仅在行尾。  
“We pruned the lies from bleeding trees / Distilled the storm from entropy”  
—— “lies/trees” 与 “trees/entropy” 形成句内回声。

**格律（Meter）**：重音与非重音音节的节奏。
- 平行行之间音节数一致有助于提升可唱性
- **重音音节**比总音节数更重要
- 大声朗读。若磕绊，说明格律需调整
- 有意打破格律可制造强调或惊喜感

## 3. 情感弧线与动态变化

将歌曲视为一段旅程，而非平坦道路。

**能量映射**（粗略参考，非硬性规定）：
  前奏：2-3  |  主歌：5-6  |  预副歌：7  
  副歌：8-9  |  桥段：视情况而定  |  最终副歌：9-10

最强大的动态技巧：**对比**。
- 在尖叫前低语，比单纯尖叫更具冲击力
- 稀疏对密集，慢对快，低对高
- “骤降”效果之所以成立，是因为有铺垫
- 沉默也是一种乐器

“从低语到咆哮，再回到低语” —— 从亲密开始，逐步推向全力，  
再剥离回脆弱。适用于民谣、史诗、颂歌。

## 4. 写出有效歌词

**展示，而非讲述**（通常如此）：
- “我很伤心” = 平淡
- “你的连帽衫还挂在门边的钩子上” = 生动
- 但有时直白说出“我愿献出生命”恰恰是力量所在

**钩子（Hook）**：
- 人们记住、哼唱、重复的那一句
- 通常是标题或核心短语
- 当旋律 + 歌词 + 情感三者对齐时效果最佳
- 将其置于最具冲击力的位置（通常是副歌首行或末行）

**词曲协调（Prosody）** —— 歌词与音乐相互支撑：
- 稳定情绪（解决、平静）搭配平稳旋律、完全押韵、解决和弦
- 不稳定情绪（渴望、怀疑）搭配游移旋律、近似押韵、未解决和弦
- 主歌旋律通常较低，副歌更高
- 但若服务于歌曲，也可反过来

**避免**（除非你有意为之）：
- 无意识使用陈词滥调（如未铺垫就使用“金子般的心”）
- 为押韵强行改变语序（“尤达式说话”）
- 每个段落能量相同（动态扁平）
- 将初稿视为神圣 —— 修改即创作

## 5. 恶搞与改编

用新歌词重写现有歌曲时：

**结构骨架**：先映射原曲结构。
- 计算每行音节数
- 标记押韵模式（ABAB、AABB 等）
- 标出哪些音节是**重音**
- 注意长音/持续音出现的位置

**适配新词**：
- 将重音音节与原曲相同节拍对齐
- 总音节数可灵活增减 1-2 个非重音音节
- 在长音上，尽量匹配原曲的**元音音色**  
  （若原曲长音为“LOOOVE”的“oo”音，“FOOOD”比“LIFE”更合适）
- 关键位置使用单音节词替换可保持节奏完整  
  （Crime → Code，Snake → Noose）
- 将新词唱在原曲上 —— 若磕绊，则需修改

**概念**：
- 选择一个足够支撑整首歌的强概念
- 从标题/钩子出发向外扩展
- **首先生成大量原始素材**（双关语、短语、意象），  
  再将最佳部分填入结构
- 若某处需要特定句子，可反向推导押韵模式以铺垫

**保留部分原曲**：保留少量原句或结构，  
可增强辨识度，让听众感受到与原作的联系。

## 6. Suno AI 提示词工程

### 风格/流派描述字段

**公式**（按需调整）：
  流派 + 情绪 + 时代 + 乐器 + 人声风格 + 制作 + 动态

```
差：  "悲伤的摇滚歌曲"
好： "电影感管弦乐间谍惊悚片，1960 年代冷战时期，烟熏感的性感女声，
      大乐队爵士，铜管组（小号与法国号）， sweeping 弦乐，小调，
      复古模拟温暖感"
```

**描述旅程，而非仅流派**：
```
以稀疏钢琴上的 haunting 低语开始。逐渐加入弱音铜管。
通过完整管弦乐在副歌中 build up。
第二主歌以 raw belting 强度爆发。
尾奏剥离回 lone piano 与 fading 至 silence 的脆弱低语。"
```

**技巧**：
- V4.5+ 支持风格字段最多 1,000 字符 —— 充分利用
- **禁止使用艺术家名称或商标**。描述声音本身。  
  “1960 年代冷战间谍惊悚片铜管”而非“詹姆斯·邦德风格”  
  “90 年代垃圾摇滚”而非“涅槃乐队风格”
- 若有偏好，请指定 BPM 与调性
- 使用“排除风格”字段说明你**不想要**的内容
- 意想不到的风格组合可能是金矿：“巴萨诺瓦陷阱”、  
  “阿巴拉契亚哥特”、“芯片爵士”
- 构建人声**人格（PERSONA）**，而非仅性别：  
  “一位饱经风霜的 torch singer，烟熏感中音，略带沙哑，
   从脆弱开始，逐步 build up 至毁灭性力量”

### 元标签（置于歌词字段内的 [方括号] 中）

**结构**：
  [前奏] [主歌] [主歌 1] [预副歌] [副歌]  
  [后副歌] [钩子] [桥段] [间奏]  
  [器乐] [器乐段落] [吉他独奏]  
  [ breakdown ] [ build-up ] [尾奏] [ silence ] [结束]

**人声表现**：
  [低语] [口语] [ belted ] [假声] [强力]  
  [灵魂感] [沙哑] [气声] [平滑] [ gritty ]  
  [断奏] [连奏] [颤音] [ melismatic ]  
  [和声] [ choir ] [和声副歌]

**动态**：
  [高能量] [低能量] [ build-up 能量] [ explosive ]  
  [情感 climax ] [ gradual swell ] [管弦乐 swell ]  
  [安静编排] [ falling tension ] [ Slow Down ]

**性别**：
  [女声] [男声]

**氛围**：
  [忧郁] [ euphoric ] [怀旧] [ aggressive ]  
  [梦幻] [ intimate ] [ dark atmosphere ]

**音效（SFX）**：
  [黑胶爆裂声] [雨] [掌声] [静电] [雷声]

**在风格字段与歌词中均放置标签以强化效果**。  
每段最多使用 5-8 个标签 —— 过多会混淆 AI。  
避免自相矛盾（同一段落中同时使用 [ calm ] 与 [ aggressive ]）。

### 自定义模式
- 严肃创作请始终使用**自定义模式**（分离“风格”与“歌词”）
- 歌词字段限制：约 3,000 字符（约 40-60 行）
- 务必添加结构标签 —— 若无，Suno 默认生成  
  扁平的主歌/副歌/主歌结构，无情感弧线

## 7. AI 歌手的语音技巧

AI 歌手不“读”词，而是“发音”。请帮助它们：

**语音重拼（Phonetic respelling）**：
- 按单词**发音**拼写：“through” → “thru”
- 专有名词失败率最高 —— 尽早测试
- “Nous” → “Noose”（强制正确发音）
- 用连字符引导音节：“Re-search”、“bio-engineering”

**演绎控制**：
- 全大写 = 更响亮、更 intense
- 元音延长：“lo-o-o-ove” = 持续音/ melisma
- 省略号：“I... need... you” = 戏剧性停顿
- 连字符拉伸：“ne-e-ed” = 情感拉伸

**务必**：
- 拼写数字：“24/7” → “twenty four seven”
- 空格分隔缩写：“AI” → “A I” 或 “A-I”
- 先在 30 秒短片段中测试专有名词/生僻词
- 一旦生成，发音即固化 —— 务必在生成前于歌词中修正

## 8. 工作流程

1. 先写出概念/钩子 —— 情感核心是什么？
2. 若为改编，先映射原曲结构（音节、押韵、重音）
3. 生成原始素材 —— 在结构化前自由 brainstorm
4. 将歌词草稿填入结构
5. 朗读/演唱 —— 发现磕绊，修正格律
6. 构建 Suno 风格描述 —— 描绘动态旅程
7. 在歌词中添加元标签以指导演绎
8. 至少生成 3-5 个变体 —— 将其视为录音 take
9. 选择最佳版本，使用“Extend/Continue”扩展有潜力的段落
10. 若意外出现精彩部分，保留它

**预期**：每获得 1 个良好结果，需约 3-5 次生成。修改是正常的。  
扩展时风格可能漂移 —— 扩展时请重申流派/情绪。

## 9. 经验教训

- 在风格字段中描述动态**弧线**远比仅列出流派重要得多。  
  “从低语到咆哮，再回到低语”为 Suno 提供了演绎地图。
- 在恶搞中保留部分原句可增强辨识度与情感重量 —— 听众能感受到原作的“幽灵”。
- 歌曲的桥段位置是转换意象的绝佳机会。  
  将原曲的具体指代替换为你主题的隐喻，  
  同时保留其情感功能（反思、转变、揭示）。
- 在钩子/标签中使用单音节词替换是保持节奏  
  同时改变含义的最 clean 方式。
- 在风格字段中对人声人格的强描述，  
  其作用大于任何单一元标签。
- 不必过分拘泥于规则。若某句打破格律但更具冲击力，就保留它。  
  感受才是关键。技艺服务于艺术，而非反之。
